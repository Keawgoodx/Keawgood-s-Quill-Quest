## 🛠 วิธีติดตั้งและเริ่มใช้งาน

1. **ติดตั้ง Python 3.8 หรือสูงกว่า** (อย่าลืมติ๊ก `Add Python to PATH` ตอนติดตั้ง)
2. **ติดตั้ง Library พื้นฐาน:** เปิด Command Prompt หรือ Terminal แล้วพิมพ์คำสั่งด้านล่าง
   ```bash
   pip install customtkinter python-docx PyMuPDF tkinterdnd2 moviepy curl_cffi requests playwright chardet beautifulsoup4
   ```
3. **ติดตั้งเบราว์เซอร์สำหรับระบบโหลดนิยาย (สำคัญมาก):**
   ```bash
   playwright install chromium
   ```
4. **เปิดโปรแกรม:** ดับเบิลคลิกไฟล์ `Keawgood_Universe.py` หรือพิมพ์คำสั่งด้านล่างใน Terminal
   ```bash
   python Keawgood_Universe.py
   ```

## 🚀 แนะนำฟีเจอร์ทั้ง 6 โมดูล

โปรแกรมนี้ประกอบไปด้วย 6 เครื่องมือหลัก คุณสามารถเปิดใช้งานพร้อมกันได้หลายหน้าต่าง:

### 1. ⬇ NOVEL (Auto Novel Downloader)
ระบบโหลดนิยายอัตโนมัติ (รองรับโหมด TWKAN Hybrid)
* **วิธีใช้:** วาง URL หน้าแนะนำนิยายหรือหน้าสารบัญ ➔ กด **"🔍 ดึงรายการตอน"** ➔ ระบุช่วงตอนที่ต้องการโหลด ➔ กด **"⚡ เริ่มโหลด"**
* **จุดเด่น:** มีปุ่ม **"🛡 ขอ Cloudflare Cookie"** เพื่อช่วยทะลวงเว็บที่ติดระบบป้องกัน Cloudflare และสามารถตั้งค่ารูปแบบชื่อไฟล์ได้เอง (เช่น `ตอนที่ [n]`)

### 2. ⧉ FILES (Novel File Manager)
ผู้ช่วยจัดการไฟล์นิยาย (รวม, แยกตอน, แปลงไฟล์)
* **วิธีใช้:** เลือกโฟลเดอร์ต้นทางและปลายทาง แล้วเลือกฟังก์ชันที่ต้องการ:
  * **รวมไฟล์:** จับไฟล์ย่อยๆ มารวมเป็นไฟล์เดียว
  * **รวมกลุ่ม:** รวมไฟล์ทีละ N ไฟล์ (เช่น รวมทีละ 5 ตอน)
  * **แยกตอน:** แตกไฟล์นิยายไฟล์ใหญ่ ออกเป็นไฟล์ย่อยทีละตอน (อ่านจากคำว่า "ตอนที่")
  * **แปลงไฟล์:** สลับนามสกุลไฟล์ไปมา (TXT, MD, DOCX) รองรับการอ่านไฟล์ PDF ด้วย

### 3. ✦ VOCAB (Vocab Optimizer)
เครื่องมือจัดการคำศัพท์นิยาย ลบคำซ้ำ
* **วิธีใช้:** ลากไฟล์ `.txt` ที่เก็บคำศัพท์มาวางในช่องซ้ายมือ ➔ กดปุ่ม **"⚡ ตัดคำซ้ำ"**
* **จุดเด่น:** เลือกระบบการเก็บคำซ้ำได้ (เก็บอันยาวสุด/เก็บอันแรก/เก็บอันล่าสุด) ช่วยประหยัดเวลาจัดระเบียบคำศัพท์

### 4. ▶ AUDIO (Audio → Video Converter)
แปลงไฟล์เสียง (Audio) และรูปภาพ ให้กลายเป็นไฟล์วิดีโอ (Batch Process)
* **วิธีใช้:** 1. ลากไฟล์เสียง `.m4a, .mp3, .wav` (ลงได้ทีละหลายไฟล์) และไฟล์รูปภาพ `.jpg, .png` 1 รูป มาวางในโปรแกรม
  2. เลือกโฟลเดอร์สำหรับเซฟวิดีโอ
  3. ระบุจำนวน "คลิปต่อ 1 ไฟล์วิดีโอ" แล้วกด **"⚡ สร้างวิดีโอ"**
* **จุดเด่น:** โปรแกรมจะเรียงลำดับไฟล์เสียงตามตัวเลขให้อัตโนมัติ (Natural Sort)

### 5. ✂ CITE (ClearCite)
เครื่องมือลบแท็กอ้างอิงอัตโนมัติ
* **วิธีใช้:** ลากไฟล์ `.txt` มาวาง ➔ เลือกโหมดว่าต้องการ **"แยกทีละไฟล์"** หรือ **"รวมเป็นไฟล์เดียว"** ➔ กด **"⚡ ประมวลผลและบันทึก"**
* **จุดเด่น:** ลบข้อความจำพวก ``, ``, `[cite_end]` ออกจากเนื้อหาให้เนียนกริบ

### 6. ✔ CHECKER (Novel Chapter Checker)
ระบบตรวจสอบความครบถ้วนของตอนนิยาย
* **วิธีใช้:** 1. ระบุ "ตอนเริ่มต้น" และ "ตอนสิ้นสุด" ที่ต้องการตรวจสอบ (เช่น 1 ถึง 500)
  2. ลากไฟล์หรือโฟลเดอร์นิยาย (`.txt, .docx, .md, .pdf`) มาวาง
  3. กด **"🔍 เริ่มสแกน"**
* **จุดเด่น:** โปรแกรมจะกวาดสายตาอ่านเนื้อหาข้างในไฟล์เพื่อหาเลขตอน แล้วสรุปให้ทันทีว่า "มีตอนไหนตกหล่นไปบ้าง"
